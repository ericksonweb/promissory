Site Header

A mobile-enabled, responsive site header containing a logo, Menu and optional additional content.

Schema

Twig Property Type Description Options CSS Class 
corporate
 boolean Applies alternate "Corporate" styling to the Site Header, featuring an Erickson Blue background and block Menu style. Use of the Corporate style is intended for single-level navigation only and will disable any dropdown submenus, which will need to be recreated elsewhere. true site-header--corporate
logo object A logo image, typically linked to the homepage of the site. 
  content
 mixed The logo image. Can be an image Media object, or inline SVG code. - logo site-header__logo 
  attributes
 object 
    href
 string The URL of the logo link. - -
menu object 
  content
 object The site navigation Menu. See Menu for syntax. site-header__menu
addl object 
  content
 mixed Additional content to display in the top-right of the header. - site-header__addl 
  hide
 boolean Hides the additional content when the Site Header is "stuck" to the top of the viewport and/or when the viewport width falls below the lg breakpoint. true site-header__addl--hide

Promissory Design

Deliver on the promise of consistent digital experiences.

Promissory Design is a library of components and styling used to create responsive, accessible, and reliable user interfaces for any Erickson Senior Living web project. Developed using atomic design principles, Promissory Design ensures predictable results made with lean, concise code.

Get Started

Include the Promissory Design CSS and JS in your HTML using the jsDelivr CDN. Then, begin building by utilizing the code samples and documentation on this site.

CSS 
<link href="https://cdn.jsdelivr.net/gh/ericksonweb/promissory@1/design/dist/css/promissory.css" rel="stylesheet" crossorigin="anonymous">
JS 
<script src="https://cdn.jsdelivr.net/gh/ericksonweb/promissory@1/design/dist/js/promissory.js" crossorigin="anonymous" defer></script>
