Icon

A small, inline SVG element generated using the Font Awesome library (v5).

Schema

Twig Property Type Description Default Options CSS Class
fa_prefix string Font Awesome style prefix. "fas" "fas", "far", "fal", "fad", "fab" [fa_prefix]
fa_name string Font Awesome icon name. "phone-alt" Search Font Awesome fa-[fa_name]
position string Position relative to related text. (See Link and Button) - "before", "after" icon--[position]
shape string Encloses the icon in a shape. - "square", "circle" icon--shape-[shape]
